# Pre Visa Hub — Official Website

Next.js 14 + Tailwind CSS marketing site for **Pre Visa Hub** (Study Abroad Consultant).

## Stack
- Next.js 14 (App Router)
- React 18
- Tailwind CSS 3
- TypeScript

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Build for production
```bash
npm run build
npm run start
```

## Replace logo with original PNG
1. Drop the original logo file at `public/logo.png`
2. Edit `src/components/Logo.tsx` to use `<Image src="/logo.png" />` instead of the inline SVG.
